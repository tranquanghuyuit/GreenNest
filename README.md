# DevSecOps Microservices Shop

Dự án môn học xây dựng website thương mại điện tử mini theo kiến trúc Microservices, có pipeline DevSecOps, containerization, Kubernetes deployment, monitoring, logging và security scanning.

## Tài Liệu Quan Trọng

- `docs/requirements.md`: yêu cầu hệ thống.
- `docs/architecture.md`: kiến trúc service.
- `docs/database.md`: thiết kế cơ sở dữ liệu.
- `docs/api.md`: thiết kế API.
- `docs/devsecops-pipeline.md`: pipeline DevSecOps.
- `docs/file-structure.md`: cấu trúc thư mục.
- `docs/threat-model.md`: mô hình rủi ro bảo mật.
- `docs/progress-log.md`: nhật ký phát triển dự án.

## Quy Tắc Làm Việc

Trước khi code, Agent cần đọc các file trong `docs/`. Khi hoàn thành task, cập nhật `docs/progress-log.md`.
